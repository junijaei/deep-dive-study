# README.md 업데이트 규칙

새 챕터 문서를 추가할 때마다 아래 3개 영역을 빠짐없이 갱신한다.

---

## 1. Progress 카운트 & 프로그레스 바

```markdown
**N / 49 챕터 완료** (XX.X%)
`████████░░░░░░░░░░░░░`
```

- `N`: 완료 챕터 수 증가
- `%`: `N / 49 * 100` 소수점 1자리
- 프로그레스 바: `█` 채움 칸 = `round(N / 49 * 21)`, 나머지 `░` (전체 21칸)

---

## 2. 주차별 진행 현황 테이블

### 변경 전

```
|      | ch01 | 프로그래밍 |      |
```

### 변경 후

```
| [Week 01](./week/week01.md) | [ch01](./chapters/ch01/README.md) | 프로그래밍 | ✅   |
```

- 주차: 해당 주차명 및 링크 추가
- 챕터: 링크 추가
- 완료: 빈칸 → `✅`

---

## 3. 챕터 목록

```markdown
### Week NN

- [chNN — 챕터 제목](./chapters/chNN/README.md)
```

- 신규 주차: `### Week NN` 섹션 추가 후 링크 나열
- 기존 주차: 해당 `### Week NN` 섹션에 링크 추가
