# 18. 함수와 일급 객체

## 18.1. 일급 객체

- 다음의 조건을 만족하는 객체를 일급 객체라고 한다.
    1. 무명의 리터럴로 생성할 수 있다. 즉, 런타임에 생성이 가능하다.
    2. 변수나 자료구조에 저장할 수 있다.
    3. 함수의 매개변수에 전달할 수 있다.
    4. 함수의 반환값으로 사용할 수 있다.
- 함수는 일급 객체다.

## 18.2. 함수 객체의 프로퍼티

- `arguments`, `caller`, `length`, `name`, `prototype`

### 18.2.1. arguments

- 함수 호출 시 전달된 인수 정보를 담고 있는 유사 배열 객체
- 자바스크립트는 매개변수와 인수의 개수가 일치하는지 확인하지 않는다.
    - 인수가 전달되지 않았을 경우 매개변수는 undefined로 초기화된다.
    - 인수가 더 많이 전달되었을 경우 무시된다.
- 모든 인수는 arguments 객체의 프로퍼티로 보관된다.
- arguments는 유사 배열 객체이자 이터러블이다.
    - 유사 배열 객체이기 때문에 배열 메서드를 바로 사용할 수 없다.
    - 이를 해결하기 위해 ES6에 Rest 파라미터가 도입됐다.

### 18.2.3. length

- 선언한 **매개변수**의 개수를 가리킨다.

### 18.2.4. name

- 함수 이름을 나타낸다.
- 익명함수 표현식의 경우
    - ES5: 빈 문자열
    - ES6: 함수 객체를 가리키는 식별자

### 18.2.5. __proto__ 접근자

- 모든 객체는 `[[Prototype]]` 이라는 내부 슬롯을 갖는다.
- __ptoto__는 프로토 타입 객체에 접근하기 위해 사용하는 접근자 프로퍼티다.

### 18.2.6. prototype 프로퍼티

- constructor만이 prototype 프로퍼티를 소유한다.
- 생성자 함수가 생성할 인스턴스의 프로토타입 객체를 가리킨다.
